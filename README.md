# Mixio

Mixio is a sleek Windows volume mixer (forked from DeejNG) that works with an Arduino Nano-based controller for hands-on control of individual app volumes.

![Image](https://github.com/user-attachments/assets/13515ab7-ffbb-4295-bf9f-3944f332c400)

Select the COM port, double-click a channel to assign an app. Mixio will auto-configure the number of sliders based on the received data.
